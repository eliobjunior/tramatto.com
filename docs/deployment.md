# Estratégia de deploy

## Hospedagem recomendada
A estrutura atual é compatível com hospedagem estática simples, como Netlify, Vercel, GitHub Pages ou qualquer servidor com suporte a arquivos HTML/CSS/JS.

## Checklist mínimo
- Enviar os arquivos estáticos para o servidor.
- Garantir que a pasta assets e css estejam presentes.
- Confirmar que a URL base e os links relativos funcionem corretamente.
- Publicar a versão final após validar páginas principais e 404.
