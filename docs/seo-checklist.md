# Checklist SEO

- Adicionar título único em cada página.
- Garantir meta description em todas as páginas.
- Usar canonical correto.
- Manter sitemap atualizado.
- Criar URLs amigáveis e consistentes.
- Incorporar dados estruturados quando houver conteúdo relevante.
- Validar velocidade de carregamento e compressão de imagens.
